

# Rozh Salam — Premium Magician Portfolio

A cinematic, David Blaine-inspired portfolio website with dark aesthetics, bold typography, smooth animations, and a world-class performer feel.

## Design Direction
- **Near-black background** with white/gold accent typography
- **Large serif headlines** mixed with clean sans-serif body text
- **Full-bleed cinematic imagery** using your uploaded photos
- **Subtle scroll animations** (fade-in, parallax, slide-up)
- **Mobile-first**, fast-loading, responsive design

---

## Pages & Features

### 1. Home Page
- **Full-screen hero** with your photo as background, name "ROZH SALAM", tagline "Impossible, up close.", and two CTA buttons: Watch Reel & Book Now
- **About section** — short bio with portrait photo
- **Featured videos** — 3-6 cards with thumbnails
- **Featured shows/acts** — 3-6 cards
- **Press quotes** section
- **Booking CTA block** at bottom

### 2. Videos Page
- Grid of video thumbnail cards
- Category filters: All / Street / Stage / Mentalism / TV / Shorts
- Featured reel pinned at top
- Modal lightbox for YouTube/Vimeo embeds

### 3. Shows / Signature Acts Page
- Cards with image + short description for each act
- Detail modal with gallery, longer description, event types, duration

### 4. Gallery Page
- Masonry photo layout using your uploaded images
- Lightbox viewer for full-screen browsing

### 5. Booking Page
- Clean form: Name, Email, Phone/WhatsApp, Country/City, Event Type, Date, Budget Range, Message
- Form validation with success state
- Direct contact blocks: email (booking@rozhsalam.com), WhatsApp (+964 772 517 3244), Instagram

### 6. Press / Media Kit Page
- Downloadable media kit section
- Headshots, bio text blocks, press quotes

## Navigation & Layout
- **Sticky top navigation** — dark/transparent, logo left, links right, mobile hamburger menu
- **Smooth scroll** between sections on home page
- **Footer** — booking email, social links (Instagram, YouTube, TikTok), copyright

## Animations & Polish
- Scroll-triggered fade/slide animations on all sections
- Smooth page transitions
- Hover effects on cards and buttons
- Video thumbnails with play overlay

## Content
- All content stored in simple data files for easy updates
- Placeholder sample entries (6 videos, 6 acts) ready to swap with real content
- Your 10 uploaded photos used throughout the site

## SEO & Performance
- Meta titles/descriptions per page
- Open Graph tags for social sharing
- Lazy-loaded images, optimized assets
- Clean URLs: /videos, /shows, /gallery, /booking, /press
